# SensitivityStudy

Nuclear reaction-rate sensitivity study tooling and inputs for nova PPN models.

This repository contains:

- Iliadis et al. 2002 extracted data products and reference PDFs.
- Nova case inputs and PPN run templates.
- Reaction plans for one-rate-at-a-time sensitivity sweeps.
- Build/run helper scripts in `tools/`.

Generated PPN run directories and compiled executables are intentionally ignored.
The recovered `NPDATA` directory is tracked where present.
